# 🌍 EcoTwin – Your AI Sustainability Companion

> Transforming carbon awareness into meaningful climate action through AI-powered guidance, personalized planning, and an evolving digital planet.

---

## 🚀 Overview

EcoTwin is an AI-powered sustainability platform that helps individuals:

- 📊 Understand their carbon footprint
- 📈 Track environmental impact over time
- 🌱 Reduce emissions through personalized action plans
- 🤖 Learn from an intelligent AI sustainability companion
- 🌍 Visualize environmental impact through a living digital ecosystem

Unlike traditional carbon calculators, EcoTwin acts as a long-term sustainability companion that adapts, remembers, and guides users toward measurable environmental improvement.

---

## 🎯 Problem Statement

Many people want to live sustainably but struggle to:

- Understand what contributes most to their carbon footprint
- Track their daily environmental impact
- Know which actions will have the greatest effect
- Stay motivated to maintain sustainable habits

EcoTwin bridges this gap by combining AI coaching, behavior tracking, and visual environmental feedback.

---

## ✨ Key Features

### 🤖 EcoBuddy Assistant

A unified AI sustainability companion capable of:

- Carbon footprint guidance
- Personalized sustainability coaching
- Carbon-related Q&A
- Activity logging
- Goal planning
- Weekly reflections
- Progress tracking

---

### 🌱 Adaptive Carbon Action Planner

Generates personalized action plans based on:

- Transportation habits
- Energy usage
- Food choices
- Historical carbon trends

Each recommendation includes:

- Expected CO₂ reduction
- Difficulty level
- Environmental impact estimate
- Progress tracking

---

### 🌍 Planet Pulse Ecosystem

A living digital planet that evolves based on user behavior.

#### Crisis State
- Pollution
- Grey atmosphere
- Dying ecosystem

#### Recovery State
- Cleaner environment
- Emerging vegetation

#### Thriving State
- Forests
- Wildlife
- Healthy ecosystem

Users can visually see the impact of their actions over time.

---

### 📊 Carbon Footprint Analytics

Track emissions across:

- Transportation
- Energy consumption
- Food habits
- Waste management

View trends, insights, and reduction opportunities.

---

### 🔮 Future Simulator

Simulate sustainability scenarios before taking action.

Examples:

- Switching to public transport
- Reducing electricity usage
- Adopting greener food habits

See projected reductions before making changes.

---

### 📈 Weekly Sustainability Reports

Automated summaries showing:

- Carbon reduction achievements
- Progress trends
- Sustainability streaks
- Personalized recommendations

---

### 🏆 Challenges & Achievements

Gamified sustainability journey featuring:

- Challenges
- Milestones
- Achievement badges
- Long-term engagement tracking

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/naveenprabhus-dev/EcoTwin.git
cd EcoTwin
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

Create a `.env` file and add your Firebase and Gemini credentials.

### Start Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```
